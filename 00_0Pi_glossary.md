# 00 — 0Pi Glossary: Identity Expand

## Core rule

A valid identity is a measurable structure before it is a slogan, metaphor, public claim, or interpretation.

## Template contract

Every Pi-stage notebook should keep the same output pattern:

- docs/{stage}_glossary.md
- docs/{stage}_notes.md
- docs/{stage}_readme_snippet.md
- results/{stage}_metadata.json
- results/{stage}_claims.json
- results/{stage}_claims.csv
- results/{stage}_manifest.json
- figures/{stage}_*.png
- math/{stage}_*.tex

## Generic terms

| TPL / CGCS term | Generic meaning | Later translations |
|---|---|---|
| identity | measurable structure | photon, molecule, field, state, cycle variable |
| expand | define without collapse | initial observable definition |
| valid construction / VC | measurement-grounded structure | physics/chemistry model |
| general-origin story / GOS | identity preserved through constraints | system-aware framing |
| invalid assignment / IA | interpretation without measurement | slogan collapse |
| zero-origin story / ZOS | claim pretending an unsupported origin | public-language shortcut |
| constraint signal | measurable variable connecting states | angle, flux, concentration, energy, rate |
| CGCS | constraint-gated comprehension score | claim-quality score |

## Translation boundary

0Pi is grammar, not application. Do not lock 0Pi to one domain.

Translate into photons, CO2, O2, climate, chemistry, or public claims later in bridge docs and later notebooks.
