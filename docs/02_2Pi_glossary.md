# 02 — 2Pi Glossary: Identity Resist

## Core rule

A valid identity is preserved across constraint pressure.

## Identity triplet

- 0Pi: define measurable identity
- 1Pi: test identity through interaction
- 2Pi: preserve identity across constraint

## Template contract

Every Pi-stage notebook should keep the same output pattern:

- docs/{stage}_glossary.md

## Generic terms

| TPL / CGCS term | Generic meaning | Later translations |
|---|---|---|
| identity | measurable structure | photon, molecule, field, state, cycle variable |
| resist | preserve identity across constraint | continuity, conservation, invariance |
| valid construction / VC | identity stabilized across constraint | physics/chemistry model |
| general-origin story / GOS | identity preserved through constraints | system-aware framing |
| invalid assignment / IA | constraint collapses identity into interpretation | slogan collapse |
| zero-origin story / ZOS | claim pretending constraint erases identity | public-language shortcut |
| constraint signal | measurable variable that stabilizes system states | angle, flux, concentration, energy, rate |
| CGCS | constraint-gated comprehension score | claim-quality score |

## Translation boundary

2Pi is grammar, not application. Do not lock 2Pi to one domain.

Translate into photons, CO2, O2, climate, chemistry, or public claims later in bridge docs and later notebooks.
