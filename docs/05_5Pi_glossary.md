# 05 — 5Pi Glossary: Physical Resist

## Core rule

A valid physical signal remains stable across constraint pressure.

## Physical triplet

- 3Pi: expand preserved identity into measurable interaction
- 4Pi: extend physical interaction through variable change
- 5Pi: resist physical collapse by preserving signal across constraint

## Generic terms

| TPL / CGCS term | Generic meaning | Later translations |
|---|---|---|
| physical signal | measurable output of interaction | signal, flux, rate, energy, angle |
| resist | preserve signal across constraint | stability test, threshold gate |
| constraint gate | threshold for stable behavior | cosθ gate, SNR gate, tolerance bound |
| stable signal | signal preserved under constraint pressure | validated measurement |
| valid construction / VC | signal passes constraint gate | physics/chemistry model |
| general-origin story / GOS | system-aware stable response | measurement-preserving framing |
| invalid assignment / IA | unstable signal treated as stable | slogan collapse |
| zero-origin story / ZOS | claim pretending instability is valid behavior | public-language shortcut |
| CGCS | constraint-gated comprehension score | claim-quality score |

## Generic 45-degree gate

The generic stability threshold can be expressed as:

```tex
\cos(\theta) \geq \frac{1}{\sqrt{1^2 + 1^2}}
```

## Translation boundary

5Pi is grammar, not application. Do not lock 5Pi to one domain.

Translate into photons, CO2, O2, climate, chemistry, or public claims later in bridge docs and later notebooks.
