# 06 — 6Pi Glossary: Cycle Expand

## Core rule

A stable physical signal becomes a cycle variable when it participates in repeatable exchange.

## Cycle triplet

- 6Pi: expand stable physical signal into repeatable cycle exchange
- 7Pi: extend cycle exchange through rates, reservoirs, or pathways
- 8Pi: resist cycle collapse by preserving exchange under constraint

## Generic terms

| TPL / CGCS term | Generic meaning | Later translations |
|---|---|---|
| cycle | repeatable exchange among states | carbon cycle, oxygen cycle, photon cycle |
| expand | introduce repeatable exchange | first cycle variable |
| stable signal | physical signal preserved by 5Pi | measurable input to cycle |
| cycle variable | quantity tracked through exchange | concentration, flux, population, state |
| exchange | measurable transfer between states | reservoir transfer, transition, pathway |
| valid construction / VC | cycle grounded in repeatable exchange | physics/chemistry model |
| general-origin story / GOS | system-aware cycle framing | measurement-preserving cycle |
| invalid assignment / IA | single event treated as cycle | slogan collapse |
| zero-origin story / ZOS | claim pretending interpretation establishes a cycle | public-language shortcut |
| CGCS | constraint-gated comprehension score | claim-quality score |

## Translation boundary

6Pi is grammar, not application. Do not lock 6Pi to one domain.

Translate into photons, CO2, O2, climate, chemistry, or public claims later in bridge docs and later notebooks.
