# 10 — 10Pi Glossary: System Extend

## Core rule

A valid system remains coherent across scale, coupling strength, and feedback.

## System triplet

- 9Pi: expand stable cycle exchange into coordinated system structure
- 10Pi: extend system structure across scale, coupling strength, and feedback
- 11Pi: resist system collapse by preserving coordination under constraint

## Generic terms

| TPL / CGCS term | Generic meaning | Later translations |
|---|---|---|
| system | coordinated structure of interacting cycles | climate system, reaction network, field system |
| extend | vary system across measurable dimensions | scale, coupling, feedback |
| scale | level or range at which system behavior is evaluated | local, regional, planetary, deep-time |
| coupling strength | measurable intensity of connection between components | exchange coefficient, interaction strength |
| feedback | system output influencing future system behavior | positive/negative feedback, buffering |
| coherence | preserved organization across variation | stable structure, coordinated response |
| valid construction / VC | system remains coherent across variables | physics/chemistry model |
| general-origin story / GOS | system-aware extension | measurement-preserving system |
| invalid assignment / IA | feedback ignored or coupling removed | slogan collapse |
| zero-origin story / ZOS | claim pretending system behavior without feedback | public-language shortcut |
| CGCS | constraint-gated comprehension score | claim-quality score |

## Translation boundary

10Pi is grammar, not application. Do not lock 10Pi to one domain.

Translate into photons, CO2, O2, climate, chemistry, or public claims later in bridge docs and later notebooks.
