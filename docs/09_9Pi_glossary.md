# 09 — 9Pi Glossary: System Expand

## Core rule

A stable cycle becomes a system when multiple constrained exchanges coordinate.

## System triplet

- 9Pi: expand stable cycle exchange into coordinated system structure
- 10Pi: extend system structure across scale, coupling strength, and feedback
- 11Pi: resist system collapse by preserving coordination under constraint

## Generic terms

| TPL / CGCS term | Generic meaning | Later translations |
|---|---|---|
| system | coordinated structure of interacting cycles | climate system, reaction network, field system |
| expand | introduce coupled cycle structure | first system structure |
| stable cycle | cycle preserved by 8Pi | measurable cycle input |
| coupling | measurable connection between cycles | exchange coefficient, feedback, interaction |
| system structure | organized relation among stable exchanges | network, reservoir system, physical model |
| coordination | constrained interaction among cycle variables | coupled transfer, feedback alignment |
| valid construction / VC | system grounded in coupled cycles | physics/chemistry model |
| general-origin story / GOS | system-aware coordination | measurement-preserving system |
| invalid assignment / IA | isolated cycle treated as full system | slogan collapse |
| zero-origin story / ZOS | claim pretending system behavior without coupling | public-language shortcut |
| CGCS | constraint-gated comprehension score | claim-quality score |

## Translation boundary

9Pi is grammar, not application. Do not lock 9Pi to one domain.

Translate into photons, CO2, O2, climate, chemistry, or public claims later in bridge docs and later notebooks.
