# 04 — 4Pi Glossary: Physical Extend

## Core rule

A valid physical interaction remains consistent under variable change.

## Physical triplet

- 3Pi: expand preserved identity into measurable interaction
- 4Pi: extend physical interaction through variable change
- 5Pi: resist physical collapse by preserving signal across constraint

## Generic terms

| TPL / CGCS term | Generic meaning | Later translations |
|---|---|---|
| physical | measurable interaction | energy, angle, rate, flux, concentration |
| extend | probe interaction across variable change | parameter sweep, perturbation, experimental variation |
| variable | measurable parameter | E, θ, r, Φ, c, Δ |
| response | measurable system output | signal change, rate change, flux response |
| valid construction / VC | response remains consistent across variables | physics/chemistry model |
| general-origin story / GOS | system-aware variable response | measurement-preserving framing |
| invalid assignment / IA | variable change collapses into interpretation | slogan collapse |
| zero-origin story / ZOS | claim pretending a single observation defines behavior | public-language shortcut |
| CGCS | constraint-gated comprehension score | claim-quality score |

## Translation boundary

4Pi is grammar, not application. Do not lock 4Pi to one domain.

Translate into photons, CO2, O2, climate, chemistry, or public claims later in bridge docs and later notebooks.
