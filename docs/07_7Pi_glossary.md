# 07 — 7Pi Glossary: Cycle Extend

## Core rule

A valid cycle remains consistent across rates, reservoirs, and pathways.

## Cycle triplet

- 6Pi: expand stable physical signal into repeatable cycle exchange
- 7Pi: extend cycle exchange through rates, reservoirs, or pathways
- 8Pi: resist cycle collapse by preserving exchange under constraint

## Generic terms

| TPL / CGCS term | Generic meaning | Later translations |
|---|---|---|
| cycle | repeatable exchange among states | carbon cycle, oxygen cycle, photon cycle |
| extend | probe cycle across structured variation | rates, reservoirs, pathways |
| rate | speed of exchange | flux rate, reaction rate, transfer rate |
| reservoir | state/container holding cycle variable | atmosphere, ocean, field state, population |
| pathway | route of transfer between reservoirs | reaction pathway, exchange channel, transition |
| extended exchange | repeatable cycle with tracked variation | cycle model, transfer network |
| valid construction / VC | cycle remains measurable across variables | physics/chemistry model |
| general-origin story / GOS | system-aware cycle extension | measurement-preserving cycle |
| invalid assignment / IA | pathway ignored or arbitrary cycle claim | slogan collapse |
| zero-origin story / ZOS | claim pretending a cycle needs no tracked transfer | public-language shortcut |
| CGCS | constraint-gated comprehension score | claim-quality score |

## Translation boundary

7Pi is grammar, not application. Do not lock 7Pi to one domain.

Translate into photons, CO2, O2, climate, chemistry, or public claims later in bridge docs and later notebooks.
