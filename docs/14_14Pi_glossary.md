# 14 — 14Pi Glossary: Rate Resist

## Core rule

A valid rate preserves meaning under constraint pressure.

## Rate triplet

- 12Pi: expand stable system behavior into measurable rates of change
- 13Pi: extend rate across intervals, baselines, and comparisons
- 14Pi: resist rate collapse by preserving rate meaning under constraint

## Generic terms

| TPL / CGCS term | Generic meaning | Later translations |
|---|---|---|
| rate | measured change per interval | concentration change, flux rate, warming rate |
| resist | preserve rate meaning under constraint | rate stability, context preservation |
| interval | counted step or measurement span | Δn, sample step, observation interval |
| baseline | reference quantity or condition | reference concentration, reference flux |
| comparison | relation between rates or rate contexts | relative rate, normalized rate |
| stable rate | comparable rate preserved under constraint | validated rate comparison |
| context collapse | rate claim losing interval or baseline | misleading comparison |
| valid construction / VC | rate context preserved under constraint | physics/chemistry model |
| invalid assignment / IA | rate remains valid after context is lost | slogan collapse |
| CGCS | constraint-gated comprehension score | claim-quality score |

## Translation boundary

14Pi is grammar, not application. Do not lock 14Pi to one domain.

Translate into photons, CO2, O2, climate, chemistry, or public claims later in bridge docs and later notebooks.
