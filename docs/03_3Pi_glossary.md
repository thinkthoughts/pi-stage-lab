# 03 — 3Pi Glossary: Physical Expand

## Core rule

A preserved identity becomes physically meaningful through measurable interaction.

## Physical triplet

- 3Pi: expand preserved identity into measurable interaction
- 4Pi: extend physical interaction through variable change
- 5Pi: resist physical collapse by preserving signal across constraint

## Generic terms

| TPL / CGCS term | Generic meaning | Later translations |
|---|---|---|
| physical | measurable interaction | energy, angle, rate, flux, concentration |
| expand | introduce observable interaction | first physical signal |
| preserved identity | identity stabilized by 2Pi | molecule, photon, field, cycle variable |
| valid construction / VC | physical meaning grounded in measurement | physics/chemistry model |
| general-origin story / GOS | system-aware physical interaction | measurement-preserving framing |
| invalid assignment / IA | interpretation without physical interaction | slogan collapse |
| zero-origin story / ZOS | claim pretending physical behavior without measurement | public-language shortcut |
| physical signal | measurable output of interaction | signal, rate, flux, energy, angle |
| CGCS | constraint-gated comprehension score | claim-quality score |

## Translation boundary

3Pi is grammar, not application. Do not lock 3Pi to one domain.

Translate into photons, CO2, O2, climate, chemistry, or public claims later in bridge docs and later notebooks.
