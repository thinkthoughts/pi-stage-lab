# 08 — 8Pi Glossary: Cycle Resist

## Core rule

A valid cycle preserves exchange under constraint pressure.

## Cycle triplet

- 6Pi: expand stable physical signal into repeatable cycle exchange
- 7Pi: extend cycle exchange through rates, reservoirs, or pathways
- 8Pi: resist cycle collapse by preserving exchange under constraint

## Generic terms

| TPL / CGCS term | Generic meaning | Later translations |
|---|---|---|
| cycle | repeatable exchange among states | carbon cycle, oxygen cycle, photon cycle |
| resist | preserve exchange under constraint | stability, buffering, feedback resistance |
| pathway | route of transfer between reservoirs | reaction pathway, exchange channel, transition |
| bounded variation | variation remains within tolerance | stable rate, bounded flux, limited deviation |
| cycle stability | exchange remains measurable under constraint | stable climate cycle, stable reaction loop |
| valid construction / VC | cycle preserves exchange under constraint | physics/chemistry model |
| general-origin story / GOS | system-aware cycle stability | measurement-preserving cycle |
| invalid assignment / IA | untracked exchange treated as valid cycle | slogan collapse |
| zero-origin story / ZOS | claim pretending pathways are unnecessary | public-language shortcut |
| CGCS | constraint-gated comprehension score | claim-quality score |

## Translation boundary

8Pi is grammar, not application. Do not lock 8Pi to one domain.

Translate into photons, CO2, O2, climate, chemistry, or public claims later in bridge docs and later notebooks.
