# 12 — 12Pi Glossary: Rate Expand

## Core rule

A stable system becomes rate-aware when change is measured per interval.

## Rate triplet

- 12Pi: expand stable system behavior into measurable rates of change
- 13Pi: extend rate across intervals, baselines, and comparisons
- 14Pi: resist rate collapse by preserving rate meaning under constraint

## Generic terms

| TPL / CGCS term | Generic meaning | Later translations |
|---|---|---|
| rate | measured change per interval | concentration change, flux rate, warming rate |
| expand | introduce change normalized by interval | first rate signal |
| measured change | difference in measurable quantity | ΔX |
| interval | counted step or measurement span | Δn, sample step, observation interval |
| rate signal | measurable change normalized by interval | ΔX / Δn |
| valid construction / VC | rate grounded in measurement | physics/chemistry model |
| general-origin story / GOS | rate-aware system framing | measurement-preserving rate |
| invalid assignment / IA | urgency treated as rate | slogan collapse |
| zero-origin story / ZOS | claim pretending rate exists without measured change | public-language shortcut |
| CGCS | constraint-gated comprehension score | claim-quality score |

## Translation boundary

12Pi is grammar, not application. Do not lock 12Pi to one domain.

Translate into photons, CO2, O2, climate, chemistry, or public claims later in bridge docs and later notebooks.
