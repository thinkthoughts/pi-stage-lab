# 16 — 16Pi Glossary: Translation Extend

## Core rule

A valid translation remains consistent across contexts, audiences, and representations.

## Translation triplet

- 15Pi: expand measurable rate structure into mapped language
- 16Pi: extend translation across contexts, audiences, and representations
- 17Pi: resist translation collapse by preserving meaning in public language

## Generic terms

| TPL / CGCS term | Generic meaning | Later translations |
|---|---|---|
| translation | mapping from measured structure to language | climate explanation, chemistry description, public note |
| extend | preserve meaning across expression contexts | audience, diagram, prose, equation |
| context | situation or domain where translation is used | seminar, article, public note, technical doc |
| audience | group receiving the translation | scientist, student, public reader |
| representation | form of expression | words, diagram, table, equation |
| context drift | meaning changes because mapping context is lost | public confusion, misleading simplification |
| valid construction / VC | translation preserves mapping across context | grounded communication |
| general-origin story / GOS | system-aware translation extension | measurement-preserving language |
| invalid assignment / IA | audience shift redefines measured meaning | slogan collapse |
| zero-origin story / ZOS | claim pretending context is unnecessary | public-language shortcut |
| CGCS | constraint-gated comprehension score | claim-quality score |

## Translation boundary

16Pi is grammar, not application. Do not lock 16Pi to one domain.

Translate into photons, CO2, O2, climate, chemistry, or public claims later in bridge docs and later notebooks.
