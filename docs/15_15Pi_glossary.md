# 15 — 15Pi Glossary: Translation Expand

## Core rule

A valid translation preserves measurable rate structure in language.

## Translation triplet

- 15Pi: expand measurable rate structure into mapped language
- 16Pi: extend translation across contexts, audiences, and representations
- 17Pi: resist translation collapse by preserving meaning in public language

## Template contract

Every Pi-stage notebook should keep the same output pattern:

- docs/{stage}_glossary.md
- docs/{stage}_notes.md
- docs/{stage}_readme_snippet.md
- results/{stage}_metadata.json
- results/{stage}_claims.json
- results/{stage}_claims.csv

- math/{stage}_*.tex

## Generic terms

| TPL / CGCS term | Generic meaning | Later translations |
|---|---|---|
| translation | mapping from measured structure to language | climate explanation, chemistry description, public note |
| expand | introduce mapped language expression | first structured translation |
| mapping | defined correspondence between quantities and terms | variables to terms |
| language expression | structured description of measurement | sentence, note, caption, abstract |
| measured structure | quantities, intervals, baselines, comparisons | ΔX, Δn, B, C |
| valid construction / VC | language preserves measurement structure | grounded communication |
| general-origin story / GOS | system-aware translation | measurement-preserving language |
| invalid assignment / IA | interpretation without measurable mapping | slogan collapse |
| zero-origin story / ZOS | claim pretending language needs no measurement | public-language shortcut |
| CGCS | constraint-gated comprehension score | claim-quality score |

## Translation boundary

15Pi is grammar, not application. Do not lock 15Pi to one domain.

Translate into photons, CO2, O2, climate, chemistry, or public claims later in bridge docs and later notebooks.
