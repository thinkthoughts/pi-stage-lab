# 17 — 17Pi Glossary: Translation Resist

## Core rule

A valid public translation preserves measured meaning under interpretation pressure.

## Translation triplet

- 15Pi: expand measurable rate structure into mapped language
- 16Pi: extend translation across contexts, audiences, and representations
- 17Pi: resist translation collapse by preserving meaning in public language

## Generic terms

| TPL / CGCS term | Generic meaning | Later translations |
|---|---|---|
| public translation | measured structure expressed for public context | article, caption, talk note, bridge page |
| resist | preserve meaning under interpretation pressure | public-language stability |
| measured meaning | recoverable quantities, context, and representation | mapped variables and terms |
| interpretation pressure | pressure to replace measurement with persuasive framing | slogan, simplification, ideology |
| recoverability | ability to map language back to measured structure | inverse mapping |
| representation boundary | distinction between expression form and measured structure | diagram vs data, prose vs equation |
| valid construction / VC | translation preserves measured meaning | grounded communication |
| general-origin story / GOS | system-aware public translation | measurement-preserving language |
| invalid assignment / IA | persuasive framing replaces measured structure | slogan collapse |
| zero-origin story / ZOS | claim pretending recoverability is unnecessary | public-language shortcut |
| CGCS | constraint-gated comprehension score | claim-quality score |

## Translation boundary

17Pi is grammar, not application. Do not lock 17Pi to one domain.

Translate into photons, CO2, O2, climate, chemistry, or public claims later in bridge docs and later notebooks.
