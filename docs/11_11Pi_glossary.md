# 11 — 11Pi Glossary: System Resist

## Core rule

A valid system preserves coordination under constraint pressure.

## System triplet

- 9Pi: expand stable cycle exchange into coordinated system structure
- 10Pi: extend system structure across scale, coupling strength, and feedback
- 11Pi: resist system collapse by preserving coordination under constraint

## Generic terms

| TPL / CGCS term | Generic meaning | Later translations |
|---|---|---|
| system | coordinated structure of interacting cycles | climate system, reaction network, field system |
| resist | preserve system coordination under constraint | stability test, feedback bound |
| system stability | preserved coordination under constraint | stable climate system, stable network |
| bounded feedback | feedback variation remains within tolerance | stable feedback, limited amplification |
| preserved coupling | measurable connections remain intact | exchange coefficient, interaction channel |
| coordination collapse | loss of measurable system structure | runaway, decoupling, model failure |
| valid construction / VC | system remains coherent under constraint | physics/chemistry model |
| general-origin story / GOS | system-aware stability | measurement-preserving system |
| invalid assignment / IA | untracked feedback treated as valid | slogan collapse |
| zero-origin story / ZOS | claim pretending coordination is unnecessary | public-language shortcut |
| CGCS | constraint-gated comprehension score | claim-quality score |

## Translation boundary

11Pi is grammar, not application. Do not lock 11Pi to one domain.

Translate into photons, CO2, O2, climate, chemistry, or public claims later in bridge docs and later notebooks.
